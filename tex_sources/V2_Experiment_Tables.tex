% ── What each group measures ─────────────────────────────────────────────
\textbf{Group A} compares all six quasi-Newton variants at identical settings — the paper's central result. On Burgers, SSBroyden reaches $\ell_2\!\sim\!10^{-8}$ vs.\ BFGS ${\sim}10^{-5}$ and L-BFGS ${\sim}10^{-3}$~\cite[Tab.~2]{kiyani2025}. We test whether these gains persist under CH's fourth-order stiffness.
%
\textbf{Group B} varies Adam warmup duration. The paper uses 1K for Burgers, 5K for Allen-Cahn~\cite[Tab.~7]{kiyani2025}; the CH biharmonic may need longer preconditioning.
%
\textbf{Group C} tests network capacity. Our v1 suite gave $\ell_2\!\sim\!O(1)$ with 1,361 params, suggesting insufficient expressiveness. The paper uses 8$\times$20 for Burgers Case~4 and 5$\times$30 for GL~\cite[Tabs.~2,~10]{kiyani2025}.
%
\textbf{Group D} is the Adam-only baseline at the same total iteration budget — the floor quasi-Newton must beat.
%
\textbf{Group E} (novel) replaces Adam with RAdam~\cite{liu2020radam}, which rectifies the adaptive LR variance, acting as built-in warmup. Not tested in~\cite{kiyani2025}.
%
\textbf{Group F} tests collocation strategy: resampling frequency ($N_\text{change}$) and total points. Our v1 found $N_\text{change}\!=\!200$ best; F3 doubles resolution to test $\nabla^4$ under-sampling.
%
\textbf{Group G} tests loss conditioning: Nocedal-Wright $H_0$ scaling~\cite[Sec.~2.3]{kiyani2025} and the power transform $\tilde{\mathcal{L}}=\mathcal{L}^{1/p}$~\cite[Sec.~2.4]{kiyani2025} that flattens the landscape near sharp minima.

\vspace{0.5em}

% ── Evaluation ───────────────────────────────────────────────────────────
Each experiment is evaluated against a spectral reference (ETDRK4, $128\!\times\!128$, $\Delta t\!=\!10^{-3}$) using four metrics:
\textbf{(1)}~relative $\ell_2$ error $e = \|U_\text{PINN} - U_\text{ref}\|_2 / \|U_\text{ref}\|_2$ on 101 snapshots;
\textbf{(2)}~mass conservation $\Delta M = \max_t |\langle U(t)\rangle - \langle U(0)\rangle|$;
\textbf{(3)}~final training loss (total and per-component);
\textbf{(4)}~wall-clock time.
All reported as mean $\pm$ std over 3 seeds.
